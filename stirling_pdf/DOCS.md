# Stirling PDF Add-on

## Description

Stirling PDF is a locally hosted web application for various PDF operations. This add-on allows you to manage PDF files directly within Home Assistant.

## Installation

1. Open Home Assistant.
2. Go to **Settings > Add-ons > Add-on Store**.
3. Click on the three-dot menu and select **Repositories**.
4. Add this repository's URL.
5. Install the Stirling PDF add-on from the add-on store.

## Configuration

No configuration options are required for this add-on.
